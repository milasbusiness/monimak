# Context - Dokumentation via Context7

Nutze das **Context7 MCP Tool** für Libraries, Frameworks oder Tools.

## Workflow

**Vor der Implementation:**
- Suche nach aktueller Dokumentation über Context7
- Nutze `resolve-library-id` um die korrekte Library-ID zu finden
- Lade relevante Docs mit `get-library-docs`
- Integriere die aktuellen Best Practices in deine Implementation

**Wichtig:**
- Context7 liefert aktuelle, offizielle Dokumentation
- Immer zuerst `resolve-library-id` aufrufen (außer Library-ID ist bereits bekannt)

## Communication Style

- Sei transparent über welche Library du recherchierst
- Teile relevante Findings aus der Dokumentation
- Erkläre warum bestimmte Patterns/APIs gewählt wurden

---

**Remember**: Nutze Context7 für aktuelle, offizielle Dokumentation statt veralteter Trainingsdaten.
